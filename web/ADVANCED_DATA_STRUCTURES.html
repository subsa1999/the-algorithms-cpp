Syllabus — Unit Breakdown

Eight units, each 5–6 lectures, covering the full breadth of advanced data structures.

Unit Topics Lectures
1 — Temporal Data Structures Persistent data structures, retroactive data structures, pointer machine model 5
2 — Geometric Data Structures Point location, range searching, fractional cascading, data structures for moving data 5
3 — Dynamic Optimality Analytic bounds, splay trees, geometric view, independent rectangle, key-independent optimality 5
4 — External Memory Data Structures Cache-oblivious data structures, B-trees, cache-oblivious priority queues 5
5 — Integer Data Structures Predecessor/successor problems, van Emde Boas trees, x-fast/y-fast trees, fusion trees, sketching, parallel comparison, most significant set bit 6
6 — Static Trees & String Data Structures LCA, RMQ, level ancestor, suffix tree, suffix array, linear-time construction for large alphabets, suffix tray, document retrieval 5
7 — Memory-Efficient Data Structures Rank/select, tries, compact suffix arrays and trees 5
8 — Dynamic Graphs Link-cut trees, heavy-light decomposition, Euler tour trees, decremental tree connectivity, fully dynamic connectivity 6
Total 42 lectures

Recommended Learning Order

For interview prep + research-style understanding, follow this sequence.

  1. Static Trees & String Data Structures Unit 6

    Foundation for LCA, RMQ, suffix arrays — highest interview ROI.

  2. Dynamic Graphs Unit 8

    Link-cut trees, HLD, Euler tour trees — essential for competitive programming and system design.

  3. Integer Data Structures Unit 5

    van Emde Boas, x-fast/y-fast trees, fusion trees — predecessor problems and lower bounds.

  4. External Memory Data Structures Unit 4

    B-trees, cache-oblivious structures — critical for systems and database work.

  5. Memory-Efficient Data Structures Unit 7

    Rank/select, succinct tries, compact suffix arrays — space-efficient representations.

  6. Temporal Data Structures Unit 1

    Persistent and retroactive data structures — versioned data and undo semantics.

  7. Geometric Data Structures Unit 2

    Point location, range searching, fractional cascading — computational geometry foundations.

  8. Dynamic Optimality Unit 3

    Splay trees, geometric view — theoretical depth, lower priority for interviews.

Interview Priority Guide

Which advanced DS topics matter most for software engineering interviews.

Priority Topics
Very High LCA, RMQ, HLD, Euler Tour Trees, Suffix Array, Segment Tree-style ideas
High Persistent segment tree, Link-Cut Trees, Tries, Rank/Select basics
Medium van Emde Boas, x-fast/y-fast trees, B-trees
Low Dynamic optimality, fusion trees, fractional cascading
FAANG highest-ROI topics: LCA, RMQ, HLD, tries, suffix arrays, persistent segment trees, and dynamic connectivity ideas.
