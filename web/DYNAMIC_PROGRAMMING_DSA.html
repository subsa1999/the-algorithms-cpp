1. Core Concepts & Framework

What is DP?

Dynamic Programming is an optimization technique for problems with overlapping subproblems and optimal substructure. Instead of recomputing the same subproblems, we store their results.

The DP Thinking Framework

5-Step Recipe (use for every DP problem):
  1. Define state — What does dp[i] (or dp[i][j]) represent?
  2. Find recurrence — How does dp[i] relate to smaller subproblems?
  3. Identify base cases — What are the trivially known answers?
  4. Determine iteration order — Fill table so dependencies are computed first.
  5. Locate answer — Where in the table is the final answer?

Top-Down (Memoization) vs Bottom-Up (Tabulation)

Top-Down (Memoization)

Recursive + cache. Natural to write. Only computes needed states.

Drawback: recursion stack depth; Python has ~1000 limit by default.

Bottom-Up (Tabulation)

Iterative + table. No stack overflow. Easier to optimize space.

Drawback: must determine correct iteration order.

Top-Down Template

pythonfrom functools import lru_cache

def solve(params):
    @lru_cache(maxsize=None)
    def dp(state):
        # Base case
        if base_condition:
            return base_value
        # Recurrence
        return min/max(dp(smaller_state) + cost for choice in choices)
    return dp(initial_state)

Bottom-Up Template

pythondef solve(params):
    dp = [base_values]
    for i in iteration_order:
        for choice in choices:
            dp[i] = min/max(dp[i], dp[smaller] + cost)
    return dp[answer_location]

Space Optimization Trick

Tip: If dp[i] only depends on dp[i-1] (or the previous row), you can reduce space from O(n²) to O(n) by keeping only two rows (or even one row with careful ordering).

2. 1D DP — Linear Sequences

Pattern: dp[i] depends on dp[i-1], dp[i-2], or a scan of dp[0..i-1]

Fibonacci / Climbing Stairs

pythondef climb_stairs(n):
    if n <= 2:
        return n
    prev2, prev1 = 1, 2
    for _ in range(3, n + 1):
        prev2, prev1 = prev1, prev2 + prev1
    return prev1

House Robber

python# dp[i] = max money robbing houses 0..i
# dp[i] = max(dp[i-1], dp[i-2] + nums[i])
def rob(nums):
    prev2, prev1 = 0, 0
    for num in nums:
        prev2, prev1 = prev1, max(prev1, prev2 + num)
    return prev1

Maximum Subarray (Kadane's Algorithm)

python# dp[i] = max subarray sum ending at index i
# dp[i] = max(nums[i], dp[i-1] + nums[i])
def max_subarray(nums):
    curr = result = nums[0]
    for num in nums[1:]:
        curr = max(num, curr + num)
        result = max(result, curr)
    return result

Coin Change (Min coins)

python# dp[a] = min coins to make amount a
def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0
    for a in range(1, amount + 1):
        for c in coins:
            if c <= a:
                dp[a] = min(dp[a], dp[a - c] + 1)
    return dp[amount] if dp[amount] != float('inf') else -1

Decode Ways

python# dp[i] = number of ways to decode s[0..i-1]
def num_decodings(s):
    if not s or s[0] == '0':
        return 0
    prev2, prev1 = 1, 1
    for i in range(1, len(s)):
        curr = 0
        if s[i] != '0':
            curr += prev1
        if 10 <= int(s[i-1:i+1]) <= 26:
            curr += prev2
        prev2, prev1 = prev1, curr
    return prev1

3. 2D DP — Grid Problems

Pattern: dp[r][c] = answer for reaching cell (r, c)

Unique Paths

python# dp[r][c] = number of paths from (0,0) to (r,c)
def unique_paths(m, n):
    dp = [[1] * n for _ in range(m)]
    for r in range(1, m):
        for c in range(1, n):
            dp[r][c] = dp[r-1][c] + dp[r][c-1]
    return dp[m-1][n-1]

Minimum Path Sum

pythondef min_path_sum(grid):
    m, n = len(grid), len(grid[0])
    for r in range(m):
        for c in range(n):
            if r == 0 and c == 0:
                continue
            elif r == 0:
                grid[r][c] += grid[r][c-1]
            elif c == 0:
                grid[r][c] += grid[r-1][c]
            else:
                grid[r][c] += min(grid[r-1][c], grid[r][c-1])
    return grid[m-1][n-1]

Maximal Square

python# dp[r][c] = side length of largest square with bottom-right at (r,c)
def maximal_square(matrix):
    m, n = len(matrix), len(matrix[0])
    dp = [[0] * n for _ in range(m)]
    max_side = 0
    for r in range(m):
        for c in range(n):
            if matrix[r][c] == '1':
                if r == 0 or c == 0:
                    dp[r][c] = 1
                else:
                    dp[r][c] = min(dp[r-1][c], dp[r][c-1], dp[r-1][c-1]) + 1
                max_side = max(max_side, dp[r][c])
    return max_side ** 2

Triangle (Minimum Path)

python# Bottom-up: start from bottom row, fold upward
def minimum_total(triangle):
    dp = triangle[-1][:]
    for row in range(len(triangle) - 2, -1, -1):
        for j in range(len(triangle[row])):
            dp[j] = triangle[row][j] + min(dp[j], dp[j+1])
    return dp[0]

4. Knapsack Patterns

4a. 0/1 Knapsack

Pattern: Given items with weight & value, pick subset to maximize value within capacity. Each item used at most once.
python# dp[w] = max value achievable with capacity w
def knapsack_01(weights, values, capacity):
    dp = [0] * (capacity + 1)
    for w, v in zip(weights, values):
        for c in range(capacity, w - 1, -1):  # reverse to avoid reuse
            dp[c] = max(dp[c], dp[c - w] + v)
    return dp[capacity]

4b. Unbounded Knapsack

Pattern: Same as 0/1 but each item can be used unlimited times. Iterate capacity forward.
python# Coin Change: unbounded knapsack counting
def coin_change_ways(coins, amount):
    dp = [0] * (amount + 1)
    dp[0] = 1
    for c in coins:            # coin loop outer → combinations
        for a in range(c, amount + 1):  # forward → reuse allowed
            dp[a] += dp[a - c]
    return dp[amount]

4c. Subset Sum / Partition Equal Subset

python# Can we partition array into two subsets with equal sum?
def can_partition(nums):
    total = sum(nums)
    if total % 2:
        return False
    target = total // 2
    dp = [False] * (target + 1)
    dp[0] = True
    for num in nums:
        for j in range(target, num - 1, -1):  # 0/1: reverse
            dp[j] = dp[j] or dp[j - num]
    return dp[target]

Key Distinction: Loop Order

Critical:
• 0/1 Knapsack: inner loop goes backward (capacity → 0) to prevent reusing items.
• Unbounded: inner loop goes forward (0 → capacity) to allow reuse.
• Combinations vs Permutations: outer loop = items → combinations; outer loop = amount → permutations.

5. Subsequence DP

Longest Increasing Subsequence (LIS)

python# O(n²) DP
def lis_dp(nums):
    n = len(nums)
    dp = [1] * n  # dp[i] = length of LIS ending at i
    for i in range(1, n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

# O(n log n) Binary Search (Patience Sorting)
import bisect
def lis_fast(nums):
    tails = []  # tails[i] = smallest tail of IS of length i+1
    for num in nums:
        pos = bisect.bisect_left(tails, num)
        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num
    return len(tails)

Longest Common Subsequence (LCS)

python# dp[i][j] = LCS length of s1[0..i-1] and s2[0..j-1]
def lcs(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    return dp[m][n]

Longest Common Substring

python# dp[i][j] = length of common substring ending at s1[i-1], s2[j-1]
def longest_common_substring(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    result = 0
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
                result = max(result, dp[i][j])
    return result

6. String DP

Edit Distance (Levenshtein)

python# dp[i][j] = min edits to convert word1[0..i-1] → word2[0..j-1]
def min_distance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1): dp[i][0] = i
    for j in range(n + 1): dp[0][j] = j

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = 1 + min(
                    dp[i-1][j],     # delete
                    dp[i][j-1],     # insert
                    dp[i-1][j-1]   # replace
                )
    return dp[m][n]

Longest Palindromic Substring

python# Expand-around-center: O(n²) time, O(1) space
def longest_palindrome(s):
    def expand(l, r):
        while l >= 0 and r < len(s) and s[l] == s[r]:
            l -= 1; r += 1
        return s[l+1:r]

    result = ""
    for i in range(len(s)):
        for p in [expand(i, i), expand(i, i+1)]:
            if len(p) > len(result):
                result = p
    return result

Palindromic Substrings Count

pythondef count_palindromes(s):
    count = 0
    def expand(l, r):
        nonlocal count
        while l >= 0 and r < len(s) and s[l] == s[r]:
            count += 1; l -= 1; r += 1
    for i in range(len(s)):
        expand(i, i)
        expand(i, i + 1)
    return count

Wildcard / Regex Matching

python# dp[i][j] = does s[0..i-1] match p[0..j-1]?
def is_match_wildcard(s, p):
    m, n = len(s), len(p)
    dp = [[False] * (n + 1) for _ in range(m + 1)]
    dp[0][0] = True
    for j in range(1, n + 1):
        if p[j-1] == '*':
            dp[0][j] = dp[0][j-1]

    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if p[j-1] == '*':
                dp[i][j] = dp[i-1][j] or dp[i][j-1]
            elif p[j-1] == '?' or s[i-1] == p[j-1]:
                dp[i][j] = dp[i-1][j-1]
    return dp[m][n]

7. Partition / MCM DP

Pattern: Split array/string at every possible point k, solve left & right, combine

Matrix Chain Multiplication (MCM)

python# dp[i][j] = min cost to multiply matrices i..j
def mcm(dims):
    n = len(dims) - 1  # number of matrices
    dp = [[0] * n for _ in range(n)]

    for length in range(2, n + 1):  # chain length
        for i in range(n - length + 1):
            j = i + length - 1
            dp[i][j] = float('inf')
            for k in range(i, j):
                cost = dp[i][k] + dp[k+1][j] + dims[i] * dims[k+1] * dims[j+1]
                dp[i][j] = min(dp[i][j], cost)

    return dp[0][n-1]

Burst Balloons

python# dp[i][j] = max coins from bursting all balloons in (i, j) exclusive
def max_coins(nums):
    nums = [1] + nums + [1]
    n = len(nums)
    dp = [[0] * n for _ in range(n)]

    for length in range(2, n):
        for left in range(n - length):
            right = left + length
            for k in range(left + 1, right):  # k = last to burst
                dp[left][right] = max(
                    dp[left][right],
                    dp[left][k] + dp[k][right] +
                    nums[left] * nums[k] * nums[right]
                )

    return dp[0][n-1]

Minimum Cost to Cut a Stick

pythondef min_cost(n, cuts):
    cuts = sorted([0] + cuts + [n])
    m = len(cuts)
    dp = [[0] * m for _ in range(m)]

    for length in range(2, m):
        for i in range(m - length):
            j = i + length
            dp[i][j] = float('inf')
            for k in range(i + 1, j):
                dp[i][j] = min(dp[i][j],
                    dp[i][k] + dp[k][j] + cuts[j] - cuts[i])

    return dp[0][m-1]

8. Interval DP

Pattern: dp[i][j] = answer for the subarray/substring from index i to j

Iteration: Always iterate by length (short → long). For each length, iterate over all starting indices.

Longest Palindromic Subsequence (Interval DP approach)

pythondef lps(s):
    n = len(s)
    dp = [[0] * n for _ in range(n)]
    for i in range(n):
        dp[i][i] = 1

    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            if s[i] == s[j]:
                dp[i][j] = dp[i+1][j-1] + 2
            else:
                dp[i][j] = max(dp[i+1][j], dp[i][j-1])

    return dp[0][n-1]

Stone Game

python# dp[i][j] = max score difference (current player - opponent) for piles[i..j]
def stone_game(piles):
    n = len(piles)
    dp = [[0] * n for _ in range(n)]
    for i in range(n):
        dp[i][i] = piles[i]

    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            dp[i][j] = max(piles[i] - dp[i+1][j],
                            piles[j] - dp[i][j-1])

    return dp[0][n-1] > 0

9. State Machine DP

Pattern: Track discrete states (e.g., holding / not holding stock; cooldown)

Best Time to Buy and Sell Stock (All Variants)

python# General framework for stock problems:
# dp[i][k][0] = max profit on day i, with k transactions left, NOT holding
# dp[i][k][1] = max profit on day i, with k transactions left, HOLDING

# ─── At most K transactions (LC 188) ───
def max_profit_k(k, prices):
    if not prices:
        return 0
    n = len(prices)
    if k >= n // 2:  # unlimited transactions
        return sum(max(0, prices[i+1] - prices[i]) for i in range(n-1))

    dp = [[[0] * 2 for _ in range(k+1)] for _ in range(n)]
    for j in range(k+1):
        dp[0][j][1] = -prices[0]

    for i in range(1, n):
        for j in range(1, k+1):
            dp[i][j][0] = max(dp[i-1][j][0], dp[i-1][j][1] + prices[i])
            dp[i][j][1] = max(dp[i-1][j][1], dp[i-1][j-1][0] - prices[i])

    return dp[n-1][k][0]

Stock with Cooldown

python# States: hold, sold (cooldown next day), rest
def max_profit_cooldown(prices):
    hold, sold, rest = -float('inf'), 0, 0
    for p in prices:
        prev_hold = hold
        hold = max(hold, rest - p)       # buy (can only buy after rest)
        rest = max(rest, sold)            # do nothing
        sold = prev_hold + p              # sell
    return max(sold, rest)

Stock with Transaction Fee

pythondef max_profit_fee(prices, fee):
    hold, cash = -prices[0], 0
    for p in prices[1:]:
        hold = max(hold, cash - p)
        cash = max(cash, hold + p - fee)
    return cash

10. Bitmask DP

Pattern: Use a bitmask integer to represent which items have been selected/visited

When: n ≤ 20 (since 2²⁰ ≈ 10⁶ states). Common for assignment, TSP, and permutation problems.

TSP — Travelling Salesman Problem

python# dp[mask][i] = min cost to visit cities in mask, ending at city i
def tsp(dist, n):
    INF = float('inf')
    dp = [[INF] * n for _ in range(1 << n)]
    dp[1][0] = 0  # start at city 0

    for mask in range(1 << n):
        for u in range(n):
            if dp[mask][u] == INF:
                continue
            if not (mask >> u) & 1:
                continue
            for v in range(n):
                if (mask >> v) & 1:
                    continue
                new_mask = mask | (1 << v)
                dp[new_mask][v] = min(dp[new_mask][v],
                                       dp[mask][u] + dist[u][v])

    full = (1 << n) - 1
    return min(dp[full][i] + dist[i][0] for i in range(n))

Minimum Cost to Assign Tasks

python# n workers, n tasks. cost[i][j] = cost for worker i to do task j
# dp[mask] = min cost to complete tasks in mask
def assign_tasks(cost, n):
    dp = [float('inf')] * (1 << n)
    dp[0] = 0

    for mask in range(1 << n):
        worker = bin(mask).count('1')  # which worker is next
        if worker >= n:
            continue
        for task in range(n):
            if not (mask >> task) & 1:
                new_mask = mask | (1 << task)
                dp[new_mask] = min(dp[new_mask],
                                    dp[mask] + cost[worker][task])

    return dp[(1 << n) - 1]

11. Digit DP

Pattern: Count numbers in [0, N] satisfying some digit property

State: dp(pos, tight, started, ...extra)
• pos = current digit position
• tight = are we still bounded by N?
• started = have we placed a non-zero digit yet? (handles leading zeros)
• Extra state depends on the constraint (digit sum, last digit, etc.)
python# Template: count numbers in [1, n] with no two adjacent equal digits
def count_special(n):
    digits = str(n)

    @lru_cache(maxsize=None)
    def dp(pos, tight, started, last_digit):
        if pos == len(digits):
            return 1 if started else 0

        limit = int(digits[pos]) if tight else 9
        count = 0

        for d in range(0, limit + 1):
            if started and d == last_digit:
                continue  # skip adjacent equal
            count += dp(
                pos + 1,
                tight and d == limit,
                started or d != 0,
                d
            )
        return count

    return dp(0, True, False, -1)

12. Tree DP

Pattern: DFS on a tree, compute dp values bottom-up from leaves

Maximum Path Sum in Binary Tree

pythondef max_path_sum(root):
    result = [-float('inf')]

    def dfs(node):
        if not node:
            return 0
        left = max(0, dfs(node.left))
        right = max(0, dfs(node.right))
        result[0] = max(result[0], left + right + node.val)
        return max(left, right) + node.val

    dfs(root)
    return result[0]

House Robber III (Tree)

python# Each node returns (rob_this, skip_this)
def rob_tree(root):
    def dfs(node):
        if not node:
            return (0, 0)
        left = dfs(node.left)
        right = dfs(node.right)
        rob = node.val + left[1] + right[1]
        skip = max(left) + max(right)
        return (rob, skip)

    return max(dfs(root))

Diameter of Binary Tree

pythondef diameter(root):

    def depth(node):
        if not node:
            return 0
        left = depth(node.left)
        right = depth(node.right)
        result[0] = max(result[0], left + right)
        return 1 + max(left, right)

    depth(root)
    return result[0]

13. DP on Graphs (DAG)

Pattern: Topological sort + relaxation. Works because DAGs have no cycles.

Longest Path in DAG

pythondef longest_path_dag(graph, n):
    # dp[u] = longest path starting from u
    dp = [-1] * n

    def dfs(u):
        if dp[u] != -1:
            return dp[u]
        dp[u] = 0
        for v, w in graph[u]:
            dp[u] = max(dp[u], dfs(v) + w)
        return dp[u]

    return max(dfs(u) for u in range(n))

Number of Paths in DAG

pythondef count_paths_dag(graph, n, src, dst):
    dp = [0] * n
    dp[dst] = 1

    order = topo_sort(graph, n)  # reverse topo order
    for u in reversed(order):
        for v in graph[u]:
            dp[u] += dp[v]

    return dp[src]

14. Game Theory DP

Pattern: Two players alternate turns. dp[state] = best outcome for current player.

Predict the Winner / Stone Game

python# dp[i][j] = max score advantage for current player from nums[i..j]
def predict_winner(nums):
    n = len(nums)
    dp = [[0] * n for _ in range(n)]
    for i in range(n):
        dp[i][i] = nums[i]

    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            dp[i][j] = max(nums[i] - dp[i+1][j],
                            nums[j] - dp[i][j-1])

    return dp[0][n-1] >= 0

Nim Game / Sprague-Grundy

python# Nim: you lose if n % 4 == 0
def can_win_nim(n):
    return n % 4 != 0

# General Sprague-Grundy
def grundy(n, moves):
    # Compute Grundy number for position n
    g = [0] * (n + 1)
    for i in range(1, n + 1):
        reachable = set()
        for m in moves:
            if i >= m:
                reachable.add(g[i - m])
        # mex: minimum excludant
        mex = 0
        while mex in reachable:
            mex += 1
        g[i] = mex
    return g[n] != 0  # first player wins if Grundy != 0

15. DP Optimizations

15a. Space Optimization (Rolling Array)

python# LCS with O(n) space instead of O(mn)
def lcs_optimized(s1, s2):
    m, n = len(s1), len(s2)
    prev = [0] * (n + 1)
    for i in range(1, m + 1):
        curr = [0] * (n + 1)
        for j in range(1, n + 1):
            if s1[i-1] == s2[j-1]:
                curr[j] = prev[j-1] + 1
            else:
                curr[j] = max(prev[j], curr[j-1])
        prev = curr
    return prev[n]

15b. Monotonic Stack / Deque Optimization

When: DP recurrence involves min/max over a sliding window. Reduces O(nk) → O(n).
python# Sliding window maximum (useful as DP optimization building block)
from collections import deque

def max_sliding_window(nums, k):
    dq = deque()  # stores indices, front = max
    result = []
    for i, num in enumerate(nums):
        while dq and nums[dq[-1]] <= num:
            dq.pop()
        dq.append(i)
        if dq[0] <= i - k:
            dq.popleft()
        if i >= k - 1:
            result.append(nums[dq[0]])
    return result

15c. Prefix Sum Optimization

python# When dp[i] = min(dp[j] + cost(j+1, i)) and cost uses a range sum,
# precompute prefix sums to get cost in O(1).
# prefix[i] = sum(nums[0..i-1])
# sum(nums[l..r]) = prefix[r+1] - prefix[l]
prefix = [0]
for num in nums:
    prefix.append(prefix[-1] + num)

15d. Convex Hull Trick (CHT)

When: Recurrence has the form dp[i] = min(dp[j] + a[i]*b[j]). Reduces O(n²) → O(n log n) or O(n).

Advanced technique — used in competitive programming for optimizing specific recurrence shapes.

15e. Divide & Conquer Optimization

When: dp[k][i] = min over j of (dp[k-1][j] + cost(j+1, i)) and the optimal split point is monotonic. Reduces O(kn²) → O(kn log n).
